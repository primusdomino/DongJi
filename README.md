# DongJi Project

## 📌 BASELINE

이 저장소는 DongJi 플랫폼의 단일 기준 문서입니다.

모든 작업은 아래 문서를 기준으로 진행합니다:

- infra/server_current_state.md → 현재 상태
- infra/server_target_architecture.md → 목표 구조
- infra/server_transition_strategy.md → 전환 전략

---

## 📁 Structure

- infra: 서버/인프라
- product: 서비스 기획
- strategy: 사업 전략
- ai: AI 시스템
- ops: 운영 기록

---

## 📌 Rule

1. 서버 변경 시 infra 문서 업데이트
2. 기능 변경 시 product 업데이트
3. 전략 변경 시 strategy 업데이트
4. 장애 발생 시 ops 기록

👉 모든 변경은 Git 기준으로 관리
