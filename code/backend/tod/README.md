# TOD Backend
기존 TOD Spring Boot 백엔드 서비스를 관리하는 영역입니다.

## Services
- tod_login
- tod_media
- tod_money
- tod_payment
- tod_shop
- tod_user
- tod_user_advert
- tod_shop_advert
- tod_franchise
- tod_batch

## Rule
- 기존 Java 서비스 구조를 유지한다.
- DB 스키마는 임의로 변경하지 않는다.
- 로그인, 회원, 매장관리 기능은 최소 수정한다.
- 신규 랭킹 기능은 별도 API 연동 방식으로 확장한다.
